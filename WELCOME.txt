Obrigado por usar meu projeto! 🚀

Este é um sistema completo de automação para LinkedIn com duas
versões poderosas:

📝 BOT SIMPLES - Para volume (100+ convites/dia)
🤖 BOT IA - Para qualidade (40-60% taxa aceitação)


⚡ INÍCIO RÁPIDO - 3 COMANDOS:
══════════════════════════════════════════════════════════════

1. Setup automático (recomendado!):
   python setup.py

2. Ou configure manualmente:
   cp config.example.py config.py
   nano config.py

3. Execute:
   python linkedin_bot.py

💡 DICAS IMPORTANTES:
══════════════════════════════════════════════════════════════

✅ FAÇA:
  • Execute setup.py primeiro
  • Comece com 10-20 convites para testar
  • Leia a documentação (vale a pena!)
  • Use delays adequados (3-7s)
  • Monitore sua conta LinkedIn

❌ NÃO FAÇA:
  • Nunca commite suas credenciais no Git
  • Não envie 100+ convites no primeiro dia
  • Não ignore warnings do LinkedIn
  • Não use sem ler a documentação
