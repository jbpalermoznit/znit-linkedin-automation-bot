# Contribuindo para o LinkedIn Automation Bot

Obrigado por considerar contribuir com este projeto! 🎉

## 📋 Código de Conduta

Este projeto segue um código de conduta. Ao participar, você concorda em manter um ambiente respeitoso e inclusivo.

## 🚀 Como Contribuir

### Reportando Bugs

Se encontrou um bug, por favor abra uma issue com:

- **Descrição clara** do problema
- **Passos para reproduzir** o bug
- **Comportamento esperado** vs comportamento atual
- **Screenshots** (se aplicável)
- **Versão** do Python e sistema operacional

### Sugerindo Melhorias

Adoramos sugestões! Abra uma issue com:

- **Descrição clara** da funcionalidade
- **Por que seria útil**
- **Exemplos de uso** (se possível)

### Pull Requests

1. **Fork** o repositório
2. **Crie uma branch** para sua feature:
   ```bash
   git checkout -b feature/MinhaNovaFeature
   ```

3. **Faça suas alterações** seguindo o estilo do código existente

4. **Teste suas mudanças:**
   ```bash
   python verificar_instalacao.py
   # Teste com poucos perfis
   ```

5. **Commit com mensagem clara:**
   ```bash
   git commit -m "Adiciona: funcionalidade X para melhorar Y"
   ```

6. **Push para sua branch:**
   ```bash
   git push origin feature/MinhaNovaFeature
   ```

7. **Abra um Pull Request** com:
   - Descrição clara das mudanças
   - Por que é necessário
   - Como foi testado

## 🎨 Estilo de Código

- Use **PEP 8** para Python
- **Comente** código complexo
- **Docstrings** para funções importantes
- **Variáveis** em português ou inglês (consistente)

## ✅ Checklist para PR

Antes de submeter, verifique:

- [ ] Código segue o estilo do projeto
- [ ] Testado com perfis reais (poucos)
- [ ] Documentação atualizada (se necessário)
- [ ] Sem credenciais no código
- [ ] `.gitignore` atualizado (se novos arquivos)
- [ ] `requirements.txt` atualizado (se novas libs)

## 🐛 Áreas que Precisam de Ajuda

- **Testes automatizados**
- **Suporte a outros navegadores** (Firefox, Edge)
- **Interface gráfica** (GUI)
- **Tratamento de captchas**
- **Melhorias na IA**
- **Tradução para inglês**
- **Docker support**

## 💡 Ideias de Contribuição

### Fácil
- Corrigir typos na documentação
- Adicionar exemplos de uso
- Melhorar mensagens de erro
- Adicionar mais templates

### Médio
- Implementar rate limiting melhor
- Adicionar analytics
- Criar dashboard de resultados
- Suporte a proxy

### Difícil
- Implementar computer vision para captchas
- Sistema de agendamento
- Integração com CRM
- A/B testing automático

## 📖 Documentação

Se adicionar funcionalidades, atualize:

- `README.md` - Visão geral
- `GUIA_TERMINAL.md` - Instruções de uso
- `GUIA_IA.md` - Funcionalidades IA
- Comentários no código

## 🧪 Testando

Sempre teste com:

1. **Poucos perfis primeiro** (5-10)
2. **Diferentes tipos de perfis**
3. **Perfis que já são conexões**
4. **Perfis privados**

## ⚠️ Segurança

**NUNCA commite:**
- Credenciais do LinkedIn
- Dados pessoais
- Logs com informações sensíveis
- Listas reais de URLs

Sempre use arquivos de exemplo!
